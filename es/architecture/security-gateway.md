# Gateway de seguridad

Defensa en profundidad alrededor de agentes, tools y datos.

Capas típicas:

1. Auth / rate limit / tenant  
2. Validación de entrada  
3. DLP / patrones sensibles  
4. Política de tools (irreversibles bloqueadas o con aprobación)  
5. Sandbox de ejecución  
6. Escaneo de salidas (inyección)  
7. Auditoría / hash chain
