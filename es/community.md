## Contribuir

```bash
git clone https://github.com/PStarH/Commander.git
cd Commander
pnpm install
pnpm test:all
```

Bienvenidos: tools, proveedores, topologías, docs, bug fixes.

Normas: tests en verde, TypeScript estricto, sin `as any` / `@ts-ignore`, commits atómicos.

Docs: [Contribuir](/es/guide/contributing).
