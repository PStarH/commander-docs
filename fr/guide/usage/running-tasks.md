# Exécuter des tâches

> CLI : `npx tsx packages/core/src/cliEntry.ts` · build : `commander`

## Usage

```bash
npx tsx packages/core/src/cliEntry.ts plan "your task"
npx tsx packages/core/src/cliEntry.ts run "your task" --stream
```
