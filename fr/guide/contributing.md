# Contribuer à la doc

Page localisée (fr) — contenu aligné sur la documentation anglaise / espagnole pour **Contribuer à la doc**.

## Entrée rapide

```bash
git clone https://github.com/PStarH/commander-docs.git
cd commander-docs
npm install
npm run dev
npm run check
npm run build
```
