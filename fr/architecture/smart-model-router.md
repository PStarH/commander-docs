# Routeur intelligent de modèles

Choisit fournisseur et modèle selon complexité de tâche, coût, latence et disponibilité (chaîne de fallback).

## En pratique

```bash
npx tsx packages/core/src/cliEntry.ts doctor
curl http://localhost:4000/health/detailed
```

Observez le comportement via le stream CLI, la console web et les métriques Prometheus.
