# Agent Transaction Runtime (ATR)

Pause, reprise et recovery transactionnels des tâches agent. Utile pour les longs runs et les redémarrages contrôlés.

## En pratique

```bash
npx tsx packages/core/src/cliEntry.ts doctor
curl http://localhost:4000/health/detailed
```
