# Gateway de sécurité

Défense en profondeur autour des agents et des tools :

1. Auth / rate limit / tenant  
2. Validation d’entrée  
3. DLP / motifs sensibles  
4. Politique des tools (irréversibles)  
5. Sandbox d’exécution  
6. Scan des sorties (injection)  
7. Audit / hash chain  

Voir aussi le [guide Sécurité](/fr/guide/security).
