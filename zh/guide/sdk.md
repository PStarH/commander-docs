# Agent SDK (TypeScript)

本页说明 Commander 中 **Agent SDK (TypeScript)** 的用途、操作方式与生产注意点。命令路径与产品 monorepo 保持一致。

## 快速入口

```bash
git clone https://github.com/PStarH/Commander.git
cd Commander && pnpm install
pnpm --filter @commander/sdk build
```
