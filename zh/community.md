## 贡献

```bash
git clone https://github.com/PStarH/Commander.git
cd Commander
pnpm install
pnpm test:all   # 或 packages/core 下的测试
```

欢迎：新工具、新提供商、拓扑、文档、bugfix。

规范：测试全绿、严格 TypeScript、无 `as any` / `@ts-ignore`、原子提交。

文档贡献：[CONTRIBUTING](/guide/contributing)（英文）。

## 安全披露
